Events in Javascript ==================== A class assignment exploring events in Javascript. Goal ---- To complete the assignment, you should: 1. Fix the bad behavior of the "Decremenent" button 2. Implement the proper behavior for the "Decrement" button. The button should appropriately modify the value on the page that is modified by the increment button. Getting Started --------------- You can open `index.html` in your web browser to run it. You can edit the file in a text editor of your choice to make modifications to the behaviors. You will need to refresh the page to see your changes.
